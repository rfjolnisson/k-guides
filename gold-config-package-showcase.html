Release Tiers: Minimize Unused Inventory Risk

You've contracted 20 rooms, but what if you only sell 12? You're paying for 8 empty rooms. Release tiers let you progressively release unused capacity back to the supplier.

Example: 20-room block, December 15 departure
60 days out (Oct 16): Release 5 rooms if unsold ‚Üí 15 remain
30 days out (Nov 15): Release 5 more ‚Üí 10 remain
14 days out (Dec 1): Release 5 more ‚Üí 5 remain
7 days out (Dec 8): Release final 5 ‚Üí 0 contracted
You only pay for rooms you couldn't sell AND didn't release in time. Smart contract management.

Common Questions

Should I create separate packages for Land-Only and Air+Land?

Yes. Create two package records.

Why: Some customers use frequent flyer points and want land-only. Others want all-inclusive pricing. Most operators see 60% book air+land, 40% book land-only. Give them both options.

Can I change minimum capacity after departures exist?

Yes, but do it carefully.

Changing minimum from 10 to 12 means departures with 11 bookings will lose their "Guaranteed" status. Make changes before the season starts, not mid-season.

How do I handle single supplements?

Configure pricing for each room type.

Single
$5,800
Double
$5,000
Twin
$5,000

System shows: "From $5,000 (double occupancy)" with +$800 single supplement clearly displayed.

What if inventory is less than my maximum capacity?

System uses the lower value.

If you set maximum capacity = 20 but only have 15 rooms available, departure shows "15 spaces left." Inventory trumps capacity setting.

Can I offer the same tour style in different regions?

Yes. Create separate packages, reuse components where possible.

"Discovery Tour" format in Japan, Egypt, Peru‚Äîsame tour style, different destinations. Create 3 packages, but payment schedules, cancellation policies, and some component types can be shared across all three.
